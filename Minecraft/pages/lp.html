Linguagem de Programação

A linguagem de programação é um conjunto de regras que define como escrever código fonte para computadores. Esse código pode ser compilado para criar programas ou interpretado diretamente para executar instruções de processamento.

O que é Java?

Java é uma linguagem de programação de alto nível, orientada a objetos e independente de plataforma, usada para desenvolver aplicações em diversos dispositivos, como computadores, servidores e dispositivos móveis. Nosso usamos durante as aulas o Intellij que é uma ferramenta de desenvolvimento de software.

Variáveis

Estrutura de decisão:

Permite que o programa tome diferentes caminhos de execução com base em condições especificadas.

If/else:permite executar um bloco de código se a condição for verdadeira, e outro bloco de código se a condição for falsa.

Switch:usada para executar um bloco de código entre várias opções possíveis com base no valor de uma expressão.

Estrutura de Repetição:

Permite que um bloco de código seja executado várias vezes, com base em uma condição.

While:continua executando um bloco de código enquanto a condição especificada for verdadeira.

Do/while:é semelhante ao while, mas garante que o bloco de código seja executado pelo menos uma vez.

For:é usado quando o número de iterações é conhecido previamente.
