Automação Industrial

Automação Industrial

A automação industrial usa tecnologias para controlar e monitorar processos, aumentando a eficiência, precisão e segurança. O Arduino, uma placa de microcontrolador acessível, facilita a criação de projetos eletrônicos interativos. Uma aplicação comum é o uso de displays LCD para exibir informações em tempo real.

Arduino

O Arduino é uma plataforma de prototipagem eletrônica de código aberto, fácil de usar em hardware e software.Usamos durante as aulas o Tikercad que é uma plataforma online, que usamos para criar e simular circuitos eletrônicos de forma prática e permitindo que a gente veja como os circuitos funcionam sem a necessidade de hardware físico.

Arduino com LCD

Utilizamos o um LCD com o Arduino que ajuda a exibir informações como leituras de sensores, status do sistema e mensagens de erro, diretamente no local de operação. LCDs são uma tecnologia de exibição encontrada em dispositivos eletrônicos como smartphones e monitores, usado para mostrar imagens e texto com uma qualidade alta e baixo consumo da energia.

Linguagem usada:

Usamos a linguagem C++ nas atividades que fizemos, já que é a linguagem principal para programar no tinkercad. C++ é a linguagem principal para o arduino.
Algumas características específicas da linguagem c++:
Função obrigatória:

setup (): Usada para começar o programa, sendo chamada uma vez.

loop (): Usada para executar o código repetidamente.

Manipulação de pinos:

pinMode(pin, mode): Configura entrada ou saída do pino.

digitalWrite(pin, value): Escreve um valor no pino digital.

digitalRead(pin): Lê o valor do pino digital.

analogRead(pin): Lê o valor do pino analógico.

analogWrite(pin, value): Escreve um valor analógico em um pino digital.

Função de Temporização:

delay(ms): pausa o programa por uma quantidade de milissegundos.

millis(): Retorna o tempo em milissegundos desde que começa a executar o programa atual.
